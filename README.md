# 74HC595-Shift-Register-With-555-And-LM358-LED-Driver
74HC595 Shift Register With 555 And LM358 LED Driver

LED chaser with 74HC595 shift register, 555 to generate clock and LM358 to reset and set the length of pulse. 

No microcontroller used.
